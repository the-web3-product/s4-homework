\## 数字货币交易所 / 中心化钱包  

\## 链上追踪 \& 反洗钱工具调研



\### 1. 场景背景



\- \*\*业务场景\*\*：数字货币交易所 / 中心化托管钱包，需要在风控系统中对链上资金进行追踪和风险识别，用于：

&nbsp; - \*\*KYT（Know Your Transaction）\*\*：识别高风险地址、交易、路径。

&nbsp; - \*\*反洗钱（AML）\*\*：发现链上洗钱路径、交易分拆、混币等行为。

&nbsp; - \*\*合规审查\*\*：满足监管机构对制裁名单、黑名单、可疑交易报告的要求。

\- \*\*追踪系统角色\*\*：作为风控系统的链上“眼睛”，核心是：

&nbsp; - 地址 \& 实体标签库

&nbsp; - 交易图谱 \& 追踪引擎

&nbsp; - 风险评分 \& 告警规则

&nbsp; - 报告与调查工具



---



\### 2. 国际主流链上追踪 / AML 服务商



\#### 2.1 Chainalysis



\- \*\*简介\*\*：全球最知名的区块链分析公司之一，为交易所、银行、执法机构提供链上分析、KYT 与调查工具。

\- \*\*优势\*\*

&nbsp; - \*\*监管与品牌认可度高\*\*：欧美多国监管机构、执法部门大量采用，利于对接欧美银行与牌照。

&nbsp; - \*\*地址标签库庞大\*\*：交易所、暗网市场、勒索软件、诈骗、Mixer 等分类标签丰富且更新快。

&nbsp; - \*\*产品线完整\*\*：包括 `KYT`（实时监控）、`Reactor`（案件追踪）、`Sanctions Screening`（制裁筛查）等。

&nbsp; - \*\*平台成熟稳定\*\*：SaaS 稳定性好，文档齐全，集成经验丰富。

\- \*\*劣势\*\*

&nbsp; - \*\*成本高\*\*：订阅费用在同类中偏高，中小型平台压力大。

&nbsp; - \*\*本地化有限\*\*：中文支持、中文合规语境下的案例相对较少。

&nbsp; - \*\*部分本地热门币种/链覆盖节奏不一定最快\*\*：尤其区域性项目。



---



\#### 2.2 TRM Labs



\- \*\*简介\*\*：近年发展快速的链上分析与合规公司，强调多链 \& DeFi 场景覆盖。

\- \*\*优势\*\*

&nbsp; - \*\*多链 \& DeFi 能力强\*\*：对跨链桥、DeFi 协议、NFT、市价稳定币等追踪较为完善。

&nbsp; - \*\*风险建模灵活\*\*：结合规则与机器学习，支持自定义风险模型与阈值，便于与自有风控引擎结合。

&nbsp; - \*\*合规视角突出\*\*：在制裁、可疑交易报告（STR）、监管报表方面支持较好。

\- \*\*劣势\*\*

&nbsp; - \*\*监管背书略弱于 Chainalysis\*\*：但在加速追赶。

&nbsp; - \*\*价格仍然偏贵\*\*：相对顶级方案略友好，但总体仍是高价位。

&nbsp; - \*\*本地化服务有限\*\*：中文材料和本地政策理解需要团队自行补充。



---



\#### 2.3 Elliptic



\- \*\*简介\*\*：欧洲老牌区块链分析公司，产品包括 `Navigator`（交易追踪）、`Lens`（地址风险）、`Discovery`（交易所风险对比）等。

\- \*\*优势\*\*

&nbsp; - \*\*在欧洲金融与监管圈认可度高\*\*：适合以欧盟、英国为主要合规目标的机构。

&nbsp; - \*\*法币网关相关风险分析较强\*\*：对接到银行、支付渠道等路径研究较深入。

&nbsp; - \*\*可视化体验良好\*\*：案件视图和图谱展示对合规与风控人员友好。

\- \*\*劣势\*\*

&nbsp; - \*\*亚洲 / 新兴市场覆盖稍弱\*\*：本地黑产、诈骗案例覆盖不如部分竞争对手。

&nbsp; - \*\*新链 / 新场景跟进速度一般\*\*：特别是新公链、NFT、部分 DeFi。

&nbsp; - \*\*价格仍属高端\*\*。



---



\#### 2.4 CipherTrace（Mastercard 旗下）



\- \*\*简介\*\*：已被 Mastercard 收购，重点将加密资产风控与传统支付体系结合。

\- \*\*优势\*\*

&nbsp; - \*\*与传统金融体系整合良好\*\*：对接银行、卡组织等有天然优势。

&nbsp; - \*\*针对 VASP 的 AML 方案成熟\*\*：有专门面向虚拟资产服务商的合规解决方案。

\- \*\*劣势\*\*

&nbsp; - \*\*产品迭代相对保守\*\*：大型金融集团体系下灵活度略低。

&nbsp; - \*\*创新场景（DeFi/NFT）覆盖也相对保守\*\*。

&nbsp; - \*\*在亚洲加密原生业务中的影响力较有限\*\*。



---



\#### 2.5 Crystal Blockchain



\- \*\*简介\*\*：源自 Bitfury 生态的分析平台，在欧洲、东欧和部分亚洲市场使用较多。

\- \*\*优势\*\*

&nbsp; - \*\*基础链上分析能力扎实\*\*：交易图谱、地址标签、可视化分析能力较完整。

&nbsp; - \*\*价格相对头部厂商更友好\*\*：适合预算有限但希望上国际产品的机构。

\- \*\*劣势\*\*

&nbsp; - \*\*全球监管顶层背书稍弱\*\*：在大型跨国银行 / 监管场景中品牌知名度不如前三家。

&nbsp; - \*\*对新链 / 新场景覆盖需要逐项验证\*\*。



---



\#### 2.6 Merkle Science / Scorechain 等



\- \*\*简介\*\*：新一代链上分析与合规厂商，主打灵活性和定制化。

\- \*\*优势\*\*

&nbsp; - \*\*定价与合作模式灵活\*\*：支持按量计费、白标等，对中小交易所友好。

&nbsp; - \*\*API 及规则引擎开放度高\*\*：利于深度集成，自建风控系统的团队可以灵活调用。

\- \*\*劣势\*\*

&nbsp; - \*\*监管与品牌影响力有限\*\*：在严监管域内需要更多解释和补充材料。

&nbsp; - \*\*数据覆盖与质量参差\*\*：不同链、不同场景的覆盖深度需实际 PoC 验证。



---



\### 3. 国内链上分析 / 追踪服务商



\#### 3.1 慢雾（SlowMist）



\- \*\*优势\*\*

&nbsp; - \*\*安全与黑产分析经验丰富\*\*：对国内链上诈骗、黑灰产手法了解深，事件分析报告在行业内认可度高。

&nbsp; - \*\*本地化强\*\*：中文服务、对本地监管语境理解更充分，对国内热门链和项目支持友好。

&nbsp; - \*\*协同应急能力好\*\*：链上攻击、资金被盗时，可以配合追踪与溯源。

\- \*\*劣势\*\*

&nbsp; - \*\*国际监管背书相对有限\*\*：在欧美银行 / 监管机构中知名度不及国际头部厂商。

&nbsp; - \*\*标准化 AML/KYT 平台能力需根据实际产品线确认\*\*：相比安全应急，AML 可能不是最早的核心产品。



---



\#### 3.2 派盾（PeckShield）



\- \*\*优势\*\*

&nbsp; - \*\*DeFi 与跨链攻击追踪能力强\*\*：对复杂攻击行为、跨链桥事件有大量实战案例。

&nbsp; - \*\*报告能力与图谱分析突出\*\*：适合处理复杂案件级别的调查。

\- \*\*劣势\*\*

&nbsp; - \*\*标准化 AML/KYT 产品布局较晚\*\*：需要与团队确认具体产品形态和接口能力。

&nbsp; - \*\*在传统金融合规语境中的积累有限\*\*：偏安全技术向。



---



\#### 3.3 OKLink、Beosin（链必安全）等



\- \*\*优势\*\*

&nbsp; - \*\*本土合规理解深\*\*：更贴近本地监管要求、语言和沟通习惯。

&nbsp; - \*\*性价比高\*\*：相对国际巨头价格更友好，适合以本地用户为主的交易所 / 钱包。

&nbsp; - \*\*中文支持与响应效率高\*\*：在接入阶段、运营阶段沟通成本低。

\- \*\*劣势\*\*

&nbsp; - \*\*国际背书有限\*\*：在境外拿牌或对接海外银行时加分有限。

&nbsp; - \*\*部分海外小众链 / 项目覆盖度需单独确认\*\*。
